# Socialite-Titan-Social - Русификатор
Инструкция по установке. 
1. Скачайте все файлы на ваш ПК в любое удобное место. 
2. Откройте папку с аддоном которая расположена \World of Warcraft\_retail_\Interface\AddOns\Socialite\i18n
3. Добавьте файл ruRU.lua
4. Откройте папку с аддоном которая расположена \World of Warcraft\_retail_\Interface\AddOns\Socialite
5. Замените файл Socialite.toc
